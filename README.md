# univer-radar — 공공데이터 대입 분석기 (라이브 데모)

대입정보포털 「어디가」가 다루지 않는 **비용(등록금·장학금)·교차분석·진로연계**에 특화한 공공데이터 대입 분석 도구입니다. (전체 버전은 비공개 저장소)

## 데모 바로가기
👉 **https://aijhmin1031-eng.github.io/univer-radar-demo/**

## 포함 기능
- **대학 지표 분석** — 신입생 충원율·등록률·중도탈락(종합 안정성 점수) × 지역·설립(국공립/사립) 교차분석 + 차트
- **등록금** — 학부 226개 대학 평균 등록금(2026) + 지역별 차트 + 홈페이지 바로가기
- **장학금·실부담** — 등록금 − 1인당 평균 장학금(재적학생 기준) · 국공립/사립 격차
- **대학 카드** — 주소·전화·홈페이지·설립일 + 등록금·재학생
- **전문대 합격선** — 학과별 합격자 평균/최저 성적 3개년(38,491) + 합격 진단 + 추세
- **지역 분석(취업률)** · **학과**(6만) · **대학 경쟁률**(2024→2025 추세) — 게시판 + 실시간 필터

> 백엔드 없이 데이터(JSON)를 브라우저에 내장해 구동합니다(최초 로드 약 16MB). 검색창 입력 즉시 필터 + 페이지네이션.
>
> ⚠️ 솔직한 한계: 4년제 학과별 합격선·성적분석·모의지원은 어디가의 제도적 비공개 데이터라 제공하지 않습니다. 본 도구는 *대체*가 아니라 어디가가 약한 **데이터 분석·비용 비교** 영역에 특화합니다.
